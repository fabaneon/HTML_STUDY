ol & ul

ol 태그는 ordered list의 약자로, 숫자나 알파벳 등 순서가 있는 목록을 만드는데 사용합니다.

ul 태그는 unordered list의 약자로, 숫자나 알파벳 등 순서가 없는 목록을 만드는데 사용합니다.

li

li 태그는 list item의 약자로, ol과 ul의 각 항목을 나열할때 사용합니다..

title

title 태그는 브라우저상에 표시되는 탭 제목을 지정합니다..
title 속성과는 다른 개체로 분류합니다.

html, head, body

html 문서작성에서의 상위 태그로 문서 본문은 body로, 그렇지 않은 내용은 head로
이 모든 문서의 내용은 html 태그로 묶어 작성합니다.
예시)
		html
			head
				!DOCTYPE html
				title 가나다라
				meta charest="utf-8"
				h1 웹사이트 제작 과제 샘플 /h1
			/head

			body
				가나다라
				가나다라
			/body
		/html

DOCTYPE

html 문서를 브라우저에게 보낼시 브라우저에게 해당 문서가 어떤 html 코드로 작성되었는지 알려주는 태그입니다.
과거에는 수많은 DOCTYPE이 있었지만 현 html5부턴 위 예시처럼 작성됩니다.





본문 정리 태그들

  1. p 태그 : 문단을 나눌 수 있는 태그, 문단별 규격은 고정되어있지만 css를 통해 커스텀이 가능합니다.
  2. br 태그 : 줄바꿈 표시 태그, 닫힘 꺽쇠를 따로 작성하지 않고 줄을 바꿀 문장 마지막에 추가해주면 됩니다.
  3. pre 태그 : pre 태그 하위로 작성된 텍스트열은 모든 띄어쓰기와 줄바꿈을 마치 워드프로세서처럼 사용하는게 가능합니다.
